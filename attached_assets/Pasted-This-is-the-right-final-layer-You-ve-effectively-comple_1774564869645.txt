3. What Still Needs Tightening (Important)

You’re very close, but there are 3 subtle issues to fix to avoid long-term drift.

3.1 Agreement Score Has a Hidden Flaw

Current:

agreement = abs(bullish - bearish) / total
Problem:

This treats:

3 bullish, 0 bearish, 3 neutral
= same as
3 bullish, 0 bearish, 0 neutral

But those are very different.

Fix:

Only count active signals:

active = bullish + bearish

if active == 0:
    agreement = 0
else:
    agreement = abs(bullish - bearish) / active * 100

This removes dilution from neutral signals.

3.2 Conviction Overweights Agreement Slightly

Right now:

agreement = 50%
confidence = 30%
strength = 20%
Problem:

A weak but aligned system can still look “high conviction”

Adjustment:
conviction =
  (agreement * 0.4) +
  (confidence * 0.3) +
  (strength * 0.3)

This balances:

alignment vs actual signal strength
3.3 You Need One More Guardrail

Right now you only block:

ACCUMULATE under low conviction

But you should also block:

TRADE_THE_HYPE under low conviction
Add:
if conviction_score < 40 and verdict == "TRADE_THE_HYPE":
    verdict = "SPECULATIVE_FLYER"

Why:

avoids false “sell signals” in noisy markets
