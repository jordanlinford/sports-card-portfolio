вӯҗ PROJECT SETUP IN REPLIT (CLEAN VERSION)
1. Create a new Repl

2. Install dependencies (clean list)

In Replit shell:

npm install express ejs express-session bcryptjs body-parser multer
npm install @prisma/client
npm install -D prisma

npm install stripe dotenv

3. Initialize Prisma

  schema.prisma
.env
