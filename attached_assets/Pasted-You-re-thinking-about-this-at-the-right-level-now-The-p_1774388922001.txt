Task 1 — Observation Layer (highest priority)
Goal

Persist every analysis result with timestamps

Implementation
Create table: card_price_observations
id (uuid, pk)
card_id (string, indexed)
price_estimate (float)
confidence (string) -- low/med/high from Gemini
raw_response (jsonb) -- optional but HIGHLY recommended
created_at (timestamp, indexed)
Modify existing flow

Wherever you currently run:

analysis = run_gemini_analysis(card)

Add:

store_observation({
  card_id,
  price_estimate,
  confidence,
  raw_response,
  created_at
})
Important constraints
Do NOT add new API calls
Do NOT change user flow
This is passive accumulation
Acceptance Criteria
Every analysis creates a row
Can query last N observations per card
No noticeable latency added

Task 3 — Batch Job (cost-controlled)

Implementation

Job flow

  analysis = run_gemini_analysis(card)
